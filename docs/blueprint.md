# **App Name**: Quizify AI

## Core Features:

- Quiz Question Generation: Generate multiple-choice quiz questions based on user-specified topic, difficulty, and starting year, ensuring accuracy with the use of a fact-checking tool. The questions should use facts available after the year specified by the user.
- Question Display: Present a multiple-choice question with four options to the user, in a clear and engaging interface.
- Answer Selection: Allow users to select an answer from the four options provided for each question.
- Feedback Mechanism: Provide immediate feedback on whether the selected answer is correct or incorrect.
- Scoring System: Tally the number of correct answers and display the final score at the end of the quiz.
- Quiz Progress: Provide an interface that shows how many questions have been answered, and the number remaining in the quiz.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to reflect intellect, knowledge, and confidence.
- Background color: Light gray (#F0F0F0) to ensure readability and reduce eye strain.
- Accent color: Yellow (#FFDA63) to highlight important elements and CTAs, ensuring high contrast.
- Body font: 'PT Sans' (sans-serif) for clear, accessible body text. Headline font: 'Space Grotesk' (sans-serif) to create visual interest, and a slightly technical feel.
- Use clear, modern icons to represent different quiz topics.
- Subtle animations and transitions to enhance user engagement, such as highlighting correct answers or smoothly moving to the next question.
